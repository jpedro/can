# Canned

A go cli tool to store encrypted goods.


## Usage

```bash
# Install canned
$ go get github.com/jpedro/canned/cli/canned

# Check the version
$ canned version

# Set the passsword
$ export CAN_PASSWORD="test"

# Start a whole new can
$ canned init

# Add some items
$ canned set hello world

# List your items
$ canned ls
```
